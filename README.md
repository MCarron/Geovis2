# Projet de géovisualisation

## Création d'une application de grimpe

Ce project  prend place dans le cadre du cours de "Géovisualisation 2" donné par le Porfesseur Mr.  Christian Kaiser à l'Université de Lausanne. L'idée de ce projet était de réaliser une application de cartographie interactive dont la thématique ce porte sur les différents lieux de grimpe que nous pouvons rencontrer dans le Chablaise Suisse. Cette dernière permet ainsi à l'utilisateur de repérer les différents lieux de grimpe présents dans cette région
